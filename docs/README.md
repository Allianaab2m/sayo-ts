# sayo-ts Documentation

sayo-ts is an opinionated convention layer on top of Effect v4 `HttpApi` — think "Rails / NestJS" for Effect v4.

Its guiding idea is that **the conventions of a codebase should be machine-readable as types and lint rules, so that humans and AI coding assistants can be held to the same standard**. Implicit conventions are the biggest obstacle for both — sayo-ts removes them.
